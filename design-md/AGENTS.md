## Design System
- Respect the DESIGN.md specification in `./DESIGN.md` — it defines colors, typography, spacing, and components
- Use the design tokens from `./assets/_tokens.css` for all color and spacing values; stick to Tailwind utilities aligned with those tokens
- Use only the fonts defined in `./fonts`; do not import or reference external typefaces
- You can find logo and image resources in `./assets`
